# Two Rooms and a Boom — Print Deck

Fan-made web tool to randomize and print character cards for
[Two Rooms and a Boom](https://tuesdayknightgames.com/) (Tuesday Knight Games).

## Use

Open `index.html` in a browser (or serve the folder):

```bash
cd two-rooms-and-a-boom
python3 -m http.server 8080
```

1. Set player count (6–30)
2. Optionally enable advanced role packs
3. Shuffle & deal
4. Print — cut along the card edges and play

## What’s included

- Legal basic deck: President, Bomber, equal Red/Blue Team, Gambler when odd
- Optional advanced packs (Doctor/Engineer, Spies, Coy Boys, greys, etc.)
- Printable card fronts, matching backs, leader cards, and hostage chart

Not an official product.
